Organizacion del proyecto Pong:

INSTALACION DEL PONG:
	WINDOWS:
		Entre a la carpeta "\PONG FINAL\Windows"
		Ejecute el archivo "PONG instalador.exe"
		Entre al TASM o DOSBOX
		Utilice el comando "PONGEXE" para ejecutar las diferentes versiones como se explica en el DataSheet.
	MAC:
		Entre a la carpeta "\PONG FINAL\Mac o Linux"
		Copie todo el contenido de esa carpeta
		Pegue el contenido en su directorio de TASM
		Entre al TASM o DOSBOX
		Utilice el comando "PONGEXE" para ejecutar las diferentes versiones como se explica en el DataSheet.

CARPETA "Librerias":
	Aqui se encuentran todas las librerias programadas por el Grupong y utilizadas para la realizacion del proyecto.
	Estas estan separadas en tres carpetas: 
		Una en la que se encuentran todos los archivos .asm con el codigo ("ASM")
		Otra en la que se encuentran todos los archivos binarios .obj para la utilizacion de las librerias sin dar el codigo. ("OBJ")
		Otra en la que se encuentran todos los archivos de inclucion .inc para incluir las librerias a nuestro programa. ("INC")

CARPETA "Pongs codigo":
	Aqui se encuentra la programacion en .asm y el archivo .exe de cada version de pong por separado.
	En el caso de que se le haya añadido una nueva libreria en el proceso de programado se encuentran en carpetas llamadas "Librerias o archivos agregados".
		Esta tiene la misma estructura que las demas carpetas añadiendo al archivo .inc para incluir las nuevas librerias.

CARPETA "Datos sobre el proyecto":
	Esta carpeta contiene toda la organizacion del proyecto:
		La bitacora
		El pryecto organizado con project
		La idea principal del proyecto antes de llevarlo a cabo

Archivo "Pong DataSheet":
	Este archivo tiene la explicacion de todas las librerias creadas y utilizadas, ademas de una descripcion de todas las versiones del pong y lo utilizado para su programacion.
